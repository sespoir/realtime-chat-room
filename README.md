# Realtime Chat Room

一个基于 React、Express 和 WebSocket 的实时公共聊天室。用户可以输入昵称进入公共房间，实时发送和接收消息。

## 本地运行

```bash
npm install
npm run dev
```

## 生产运行

```bash
npm ci
npm run build
NODE_ENV=production npm start
```

## Hugging Face Spaces

本项目使用 Docker Space，服务监听端口为 `7860`。

健康检查路径：`/api/health`
